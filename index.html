L A T I T U D E

The purpose of this project is to analyze how weather changes as you get
closer to the equator. To accomplish this, we first pulled data from the OpenWeatherMap
API to assemeble a dataset of over 500 cities.

After retreiving the data, we used Matplotlib to plot various aspects of weather vs lattitude.
Factors we looked into include lattitude vs cloudiness, tempeture, humidity and wind speed.
This site procides the source data and visuilizations created as part of the analysis,
as well as explainations and descriptions of any trends and correlations.
